# Shadow-Survival
A top-down survival game built with Godot where the player must survive endless waves of enemies that chase them across the map.

# Godot Survival Game

A simple top-down survival game built with Godot.

## Features

- Player movement (WASD)
- Enemy AI that chases the player
- Enemy spawning system
- Basic survival gameplay loop

## Controls

WASD - Move

## Built With

Godot Engine
GDScript
